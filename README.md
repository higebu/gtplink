# gtplink

## Install

```
go get github.com/higebu/gtplink
```

## Usage

```
Usage of ./gtplink:
  -mtu int
        mtu for gtp dev (default 1280)
  -name string
        gtp dev name (default "gtp0")
  -pdp-hash-size int
        pdp hash size (default 1024)
  -sgsn-mode
        sgsn mode
  -v0-addr string
        v0 addr (default "0.0.0.0:3386")
  -v1-addr string
        v1 addr (default "0.0.0.0:2152")
```
